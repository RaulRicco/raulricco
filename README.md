## 🚀 Configuração e Deploy

### Pré-requisitos

1. **Domínio:** `raulricco.com.br` configurado
2. **SSL/HTTPS:** Certificado SSL ativo (obrigatório para Google OAuth)
3. **Hospedagem:** Servidor web com suporte a HTML estático

### Passos para Deploy

1. **Upload dos Arquivos:**
   ```bash
   # Fazer upload de todos os arquivos mantendo a estrutura
   - index.html
   - sobre.html
   - contato.html
   - politica-de-privacidade.html
   - termos-de-uso.html
   - css/style.css
   - js/main.js
   - robots.txt
   - sitemap.xml
   ```

2. **Configurar HTTPS:**
   - Ativar SSL/TLS no servidor
   - Redirecionar HTTP → HTTPS
   - Verificar certificado válido

3. **Google Search Console:**
   ```html
   <!-- Adicionar meta tag de verificação no <head> de index.html -->
   <meta name="google-site-verification" content="SEU_CÓDIGO_AQUI">
   ```

4. **Configurar DNS:**
   - Apontar domínio para servidor
   - Configurar registros A e CNAME
   - Aguardar propagação (24-48h)

5. **Testar URLs:**
   - https://raulricco.com.br/
   - https://raulricco.com.br/politica-de-privacidade.html
   - https://raulricco.com.br/termos-de-uso.html
   - Verificar acessibilidade pública (sem login)

6. **Google Cloud Console:**
   - Criar projeto no Google Cloud Console
   - Configurar OAuth Consent Screen
   - Adicionar URLs de Política de Privacidade e Termos de Uso
   - Adicionar domínio autorizado
   - Submeter para verificação

### Personalizações Necessárias

⚠️ **ATENÇÃO:** Antes do deploy, substituir os seguintes placeholders:

1. **CNPJ no Footer:**
   ```html
   <!-- Em todas as páginas, no footer -->
   <p>CNPJ: [A SER PREENCHIDO]</p>
   <!-- Substituir por: -->
   <p>CNPJ: XX.XXX.XXX/XXXX-XX</p>
   ```

2. **Imagem do Hero:**
   ```html
   <!-- Em index.html, substituir placeholder por foto real -->
   <img src="https://via.placeholder.com/500x600/..." alt="Raul Ricco">
   <!-- Por: -->
   <img src="images/raul-ricco-photo.jpg" alt="Raul Ricco - Especialista em Tráfego Pago">
   ```

3. **Link Instagram:**
   ```html
   <!-- Confirmar e atualizar se necessário -->
   <a href="https://instagram.com/raulricco" ...>
   ```

4. **Google Analytics (opcional):**
   ```html
   <!-- Adicionar no <head> se desejado -->
   <script async src="https://www.googletagmanager.com/gtag/js?id=G-XXXXXXXXXX"></script>
   ```
