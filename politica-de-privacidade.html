Política de Privacidade

Última atualização: Março de 2025

A presente Política de Privacidade descreve como Raul Ricco ("nós", "nosso" ou "nos") coleta, usa, armazena e protege os dados pessoais dos visitantes e usuários do site raulricco.com.br ("Site"). Esta política está em conformidade com a Lei Geral de Proteção de Dados (LGPD - Lei nº 13.709/2018) e com as políticas de uso de dados do Google (Google API Services User Data Policy).

Ao utilizar nosso Site e nossos serviços, você concorda com os termos desta Política de Privacidade. Caso não concorde com qualquer item, pedimos que não utilize nossos serviços.

1. IDENTIFICAÇÃO DO RESPONSÁVEL PELO TRATAMENTO DE DADOS

Controlador de Dados: Raul Ricco

Site: raulricco.com.br

E-mail para questões de privacidade: contato@raulricco.com.br

Localização: Brasília - DF, Brasil

Raul Ricco é o controlador dos dados pessoais coletados através deste Site e dos serviços prestados. Para questões relacionadas à privacidade, proteção de dados ou exercício de direitos do titular, entre em contato através do e-mail acima.

2. QUAIS DADOS SÃO COLETADOS

Coletamos diferentes tipos de dados pessoais, dependendo da forma como você interage com nosso Site:

2.1. Dados Fornecidos Diretamente pelo Usuário

  • Nome completo
  • E-mail
  • Telefone/WhatsApp
  • Mensagens e informações enviadas através de formulários de contato
  • Dados fornecidos durante orçamentos ou contratação de serviços

2.2. Dados Coletados Automaticamente

Quando você visita nosso Site, coletamos automaticamente informações técnicas, incluindo:

  • Endereço IP
  • Tipo de navegador e sistema operacional
  • Páginas visitadas e tempo de permanência
  • Origem de acesso (site de referência)
  • Data e hora de acesso
  • Cookies e tecnologias semelhantes

2.3. Dados de Ferramentas de Terceiros

Utilizamos ferramentas de terceiros que podem coletar dados adicionais:

  • Google Analytics: Dados de navegação, comportamento, dispositivo e localização aproximada
  • Google Ads: Dados de interação com anúncios e conversões
  • Google Tag Manager: Gerenciamento de tags de rastreamento
  • Meta Pixel (Facebook/Instagram): Dados de interação e conversões para remarketing

3. FINALIDADE DO TRATAMENTO DOS DADOS

Os dados pessoais coletados são utilizados para as seguintes finalidades:

3.1. Prestação de Serviços

  • Responder solicitações de contato e orçamento
  • Prestar serviços de gestão de tráfego pago (Google Ads, Meta Ads, Google Meu Negócio)
  • Executar contratos de prestação de serviços
  • Emitir relatórios e comunicações sobre campanhas

3.2. Comunicação e Marketing

  • Envio de newsletters e conteúdos educativos (mediante consentimento)
  • Comunicações sobre novos serviços e ofertas
  • Remarketing e anúncios personalizados

3.3. Análise e Melhorias

  • Análise de desempenho do Site através de ferramentas como Google Analytics
  • Otimização da experiência do usuário
  • Identificação de tendências e preferências dos visitantes
  • Melhoria contínua dos nossos serviços

3.4. Cumprimento de Obrigações Legais

  • Cumprimento de obrigações fiscais, trabalhistas e regulatórias
  • Resposta a ordens judiciais e requisições de autoridades

4. BASE LEGAL PARA O TRATAMENTO DE DADOS (LGPD)

O tratamento de dados pessoais é realizado com base nas seguintes hipóteses legais previstas na LGPD:

  • Consentimento do titular (Art. 7º, I): Para envio de comunicações de marketing e uso de cookies não essenciais
  • Execução de contrato (Art. 7º, V): Para prestação dos serviços contratados
  • Legítimo interesse (Art. 7º, IX): Para análise de dados de navegação, segurança e melhorias do Site
  • Cumprimento de obrigação legal (Art. 7º, II): Para atendimento de requisitos fiscais e regulatórios

5. COMPARTILHAMENTO DE DADOS

Seus dados pessoais podem ser compartilhados nas seguintes situações:

5.1. Com Prestadores de Serviços

  • Google LLC: Google Ads, Google Analytics, Google Tag Manager, Google My Business API
  • Meta Platforms Inc.: Facebook Ads, Instagram Ads, Meta Pixel
  • Provedores de hospedagem e infraestrutura de TI

Todos os prestadores de serviços são cuidadosamente selecionados e obrigados a manter a confidencialidade e segurança dos dados, utilizando-os apenas para as finalidades autorizadas.

5.2. Não Vendemos Seus Dados

Importante: Não vendemos, alugamos ou comercializamos seus dados pessoais para terceiros em nenhuma circunstância.

5.3. Exigências Legais

Podemos divulgar dados pessoais quando exigido por lei ou por ordem judicial, ou para proteger nossos direitos legais.

6. USO DE SERVIÇOS E APIs DO GOOGLE

Esta seção é especialmente importante para conformidade com o Google Cloud Console e verificação OAuth.

6.1. Serviços Google Utilizados

Utilizamos os seguintes serviços do Google em nosso Site e prestação de serviços:

  • Google Ads API: Para criação, gestão e otimização de campanhas de anúncios pagos
  • Google Analytics: Para análise de tráfego, comportamento de usuários e métricas de desempenho do Site
  • Google Tag Manager: Para gestão centralizada de tags de rastreamento e códigos de marketing
  • Google My Business API: Para gerenciamento de perfis de negócios locais de clientes no Google

6.2. Conformidade com Google API Services User Data Policy

Nosso uso dos serviços e APIs do Google está em total conformidade com a Google API Services User Data Policy, incluindo os requisitos de uso limitado (Limited Use Requirements).

Compromissos específicos:

  • Os dados obtidos através de APIs do Google são utilizados exclusivamente para fornecer e melhorar os serviços contratados pelo usuário
  • Não transferimos, vendemos ou usamos dados de APIs do Google para fins de publicidade direcionada, além do escopo necessário para prestação dos serviços
  • Não permitimos que humanos leiam os dados obtidos via APIs Google, exceto quando:
    • Tivermos obtido consentimento explícito do usuário
    • For necessário por razões de segurança, conformidade legal ou detecção de abuso
    • Os dados forem agregados e anonimizados
  • Utilizamos os dados apenas para as finalidades declaradas nesta Política de Privacidade
  • Mantemos políticas rigorosas de segurança e criptografia para proteção dos dados

6.3. Permissões de Acesso (OAuth)

Quando você nos autoriza a acessar serviços do Google em seu nome (por exemplo, para gerenciar suas campanhas de Google Ads ou perfil do Google Meu Negócio), solicitamos apenas as permissões estritamente necessárias para prestar o serviço contratado.

Você pode revogar essas permissões a qualquer momento através de:

  • Configurações de segurança da sua Conta Google: https://myaccount.google.com/permissions
  • Solicitação direta através do e-mail: contato@raulricco.com.br

7. COOKIES E TECNOLOGIAS SEMELHANTES

7.1. O Que São Cookies

Cookies são pequenos arquivos de texto armazenados no seu dispositivo quando você visita um site. Eles permitem que o Site reconheça seu dispositivo e melhore sua experiência de navegação.

7.2. Tipos de Cookies Utilizados

  • Cookies Essenciais: Necessários para o funcionamento básico do Site (navegação, segurança)
  • Cookies de Desempenho: Coletam informações sobre como você usa o Site (Google Analytics)
  • Cookies de Funcionalidade: Lembram suas preferências e configurações
  • Cookies de Marketing: Utilizados para exibir anúncios relevantes e medir eficácia de campanhas (Google Ads, Meta Pixel)

7.3. Como Gerenciar Cookies

Você pode gerenciar ou desativar cookies através das configurações do seu navegador:

  • Google Chrome: Configurações > Privacidade e Segurança > Cookies
  • Mozilla Firefox: Opções > Privacidade e Segurança
  • Safari: Preferências > Privacidade
  • Microsoft Edge: Configurações > Cookies e Permissões de Site

Atenção: Bloquear cookies pode afetar a funcionalidade e experiência do Site.

7.4. Opt-out de Publicidade Comportamental

  • Google: https://adssettings.google.com
  • Facebook: https://www.facebook.com/ads/preferences

8. DIREITOS DO TITULAR DOS DADOS (LGPD - Art. 18)

De acordo com a LGPD, você possui os seguintes direitos sobre seus dados pessoais:

  • Confirmação e Acesso: Confirmar se tratamos seus dados e acessá-los
  • Correção: Solicitar correção de dados incompletos, inexatos ou desatualizados
  • Anonimização ou Bloqueio: Solicitar anonimização ou bloqueio de dados desnecessários ou excessivos
  • Eliminação: Solicitar eliminação de dados tratados com seu consentimento (exceto quando houver outra base legal)
  • Portabilidade: Solicitar a transferência dos seus dados a outro fornecedor de serviço
  • Informação sobre Compartilhamento: Saber com quais entidades públicas e privadas compartilhamos seus dados
  • Revogação do Consentimento: Revogar o consentimento a qualquer momento
  • Oposição: Opor-se a tratamento realizado sem seu consentimento

8.1. Como Exercer Seus Direitos

Para exercer qualquer um desses direitos, envie um e-mail para: contato@raulricco.com.br

Responderemos sua solicitação em até 15 dias úteis. Podemos solicitar informações adicionais para confirmar sua identidade antes de processar a solicitação.

9. SEGURANÇA DOS DADOS

9.1. Medidas de Segurança

Adotamos medidas técnicas e organizacionais para proteger seus dados pessoais, incluindo:

  • Criptografia de dados em trânsito (HTTPS/SSL)
  • Controle de acesso restrito a dados pessoais
  • Armazenamento seguro em servidores protegidos
  • Monitoramento contínuo de vulnerabilidades
  • Backups regulares e planos de recuperação
  • Treinamento de equipe sobre proteção de dados

9.2. Retenção de Dados

Mantemos seus dados pessoais apenas pelo tempo necessário para cumprir as finalidades descritas nesta Política, exceto quando a lei exigir retenção por período maior.

  • Dados de contato de potenciais clientes: Até 2 anos após o último contato, salvo consentimento para manutenção
  • Dados de clientes ativos: Durante a vigência do contrato e até 5 anos após o término (obrigações legais)
  • Dados de navegação (Analytics): 26 meses (padrão Google Analytics)

10. TRANSFERÊNCIA INTERNACIONAL DE DADOS

Alguns dos nossos prestadores de serviços (como Google e Meta) possuem servidores localizados fora do Brasil, podendo resultar em transferência internacional de dados. Essas empresas adotam medidas adequadas de proteção de dados conforme exigido pela LGPD e legislações internacionais (como GDPR).

11. PRIVACIDADE DE CRIANÇAS E ADOLESCENTES

Nossos serviços não são direcionados a menores de 18 anos. Não coletamos intencionalmente dados de crianças ou adolescentes. Caso identifiquemos coleta inadvertida de dados de menores, eliminaremos tais dados imediatamente.

12. ALTERAÇÕES NESTA POLÍTICA DE PRIVACIDADE

Podemos atualizar esta Política de Privacidade periodicamente para refletir mudanças em nossas práticas, serviços ou legislação aplicável.

Como você será notificado:

  • Publicaremos a versão atualizada nesta página com nova data de "Última atualização"
  • Alterações substanciais serão comunicadas por e-mail aos usuários cadastrados
  • Recomendamos que você revise esta política periodicamente

13. LEGISLAÇÃO E FORO

Esta Política de Privacidade é regida pela legislação brasileira, especialmente pela Lei Geral de Proteção de Dados (Lei nº 13.709/2018), Marco Civil da Internet (Lei nº 12.965/2014) e Código de Defesa do Consumidor (Lei nº 8.078/1990).
